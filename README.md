
    <title>Mystery Message Application - README</title>
    <h1>Mystery Message Application</h1>
    <p>
        A full-stack <strong>Mystery Message Application</strong> built with 
        <strong>Next.js</strong>. This project integrates a custom authentication system, 
        JWT tokens, session management, and AI-powered features. It is designed to provide secure and 
        interactive messaging capabilities with a user-friendly dashboard.
    </p>
    <div class="features">
        <h2>Features</h2>
        <ul>
            <li><strong>Custom Authentication:</strong> Secure user signup, login, and OTP verification.</li>
            <li><strong>Database Integration:</strong> Efficient and scalable database handling.</li>
            <li><strong>Email Notifications:</strong> OTP-based email verification using Resend.</li>
            <li><strong>AI-Powered Features:</strong> Enhance user experience with AI integration.</li>
            <li><strong>Modern UI:</strong> Built with React Hook Form, Shadcn, and responsive components.</li>
            <li><strong>Robust Validation:</strong> Using <strong>Zod</strong> for schema validation.</li>
            <li><strong>Secure Messaging:</strong> Backend API with aggregation pipelines for data handling.</li>
        </ul>
    </div>
